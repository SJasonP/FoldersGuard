# General CLI Commands

## `fg help`

Prints CLI usage.

Usage:

```text
foldersguard help
foldersguard -h
foldersguard --help
fg help
fg -h
fg --help
```

Running `foldersguard` or `fg` without arguments is equivalent to the matching help command.

## `fg version`

Prints the application id, product version, and native format version.

Usage:

```text
fg version
```

Output:

```text
app_id=com.SJasonP.FoldersGuard
product_version=1.1.0
format_version=fg-native-v1
```
